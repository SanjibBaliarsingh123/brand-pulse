# Brand Pulse – YouTube Brand Sentiment Analyzer

Brand Pulse is a Python-based project that collects comments from the top 10 YouTube videos related to a brand using YouTube Data API.

## Features

* Fetch top 10 videos for a brand
* Extract comments
* Store video_id, comment, date, and time in JSON
* Filter non-English comments

## Technologies Used

* Python
* YouTube Data API
* JSON

## Example

Input: swiggy
Output: swiggy_data.json
